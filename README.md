# E-Commerce Experience

Este es un repositorio desarrollado para curso de Buenos Aires aprende, especificamente el curso de React js 2025.

Este repositorio contiene un template "Maquetado" de como sera una E-Commerce basica, obteniendo los productos de una API "firebase" y manejando un carrito interno sobre los productos.
Ademas contiene react router, url protegida para usuario admin, pagina para detalle de producto, sistema de loguearse y registrarse.
